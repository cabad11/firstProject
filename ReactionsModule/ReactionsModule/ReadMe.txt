Данный модуль создает элемент со счетчиком реакций и заполняет им входящий элемент.
Как использовать:
1)Подключить модуль к странице();
2)Инициальзировать через new Module.ReactionsModule({
		RootElement: ссылка на корневой элемент
		text: текст который будет указан,
		arrayEmoji: массив смайликов для модуля(пример: ["😇", "😅", "😎","😎"])
		Storage?: объект или массив с количеством кликов;
        saveFunc?(key: string | number, value: string | number): void функция записи в Storage; 
	});
